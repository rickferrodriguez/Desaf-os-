Clases de Vim

Saltos de Linea

Importante algo que no me acordaba, para saltar hacia adelante existen dos formas:

  1. La primera es usando la letra 'E' y te lleva al final de cada palabra.
  2. La segunda es con la letra 'W' esta te lleva al principio de cada palabra.

Insertar Texto

Para agregar texto ademas de las teclas 'a' y 'i' existe una letra que agrega texto al final de la palabra si te encuentras por ejemplo en medio de ella, esta es la 'A' mayuscula y parece muy interesante de usar.

Eliminar Texto

Para eliminar texto se puede presionar la tecla 'x' en el modo de visualizacion. Tambien tengo entendido que se puede borrar una linea completa con la letra 'd' dos veces, en el modo de visualizacion.

Guardar Cambios

Aunque esto ya la sabia para poder guardar un cambio es necesario presionar la tecla de los dos puntos ':' y despues la 'w' si despues deseas guardar y salir puedes presionar 'wq' esto es para guardar y salir.

Moverse entre Archivos

Si deseas llegar a la definicion de cualquier objeto puedes combinar las teclas 'g' y 'd' Pero si se desea ir directamente al archivo puedes presionar las teclas 'g' y 'f' por ejemplo desde este archivo HTML hasta el archivo style.css Importante: Una vez se guarde este archivo podras navegar hacia otros archivos.

Nota:

Existe una serie de historial cuando se navega entre definiciones o archivos, por lo tanto puedes volver hacia atras con las teclas 'Control'+ 'O'

Eliminar, undo y redo

Para eliminar una palabra dentro del modo de visualizacion se puede hacer presionando la tecla 'd' y la tecla 'w' de 'palabra'

Rehacer

Para rehacer cambios se puede presionar las teclas 'control' y 'r'

Eliminar texto de una Linea completa sin borrar la linea

Se debe presionar las teclas 'd' y '$'
