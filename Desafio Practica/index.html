Clases de Vim

Saltos de Linea

Importante algo que no me acordaba, para saltar hacia adelante existen dos formas:

  1. La primera es usando la letra 'E' y te lleva al final de cada palabra.
  2. La segunda es con la letra 'W' esta te lleva al principio de cada palabra.

Insertar Texto

Para agregar texto ademas de las teclas 'a' y 'i' existe una letra que agrega texto al final de la palabra si te encuentras por ejemplo en medio de ella, esta es la 'A' mayuscula y parece muy interesante de usar.

Eliminar Texto

Para eliminar texto se puede presionar la tecla 'x' en el modo de visualizacion. Tambien tengo entendido que se puede borrar una linea completa con la letra 'd' dos veces, en el modo de visualizacion.

Guardar Cambios

Aunque esto ya la sabia para poder guardar un cambio es necesario presionar la tecla de los dos puntos ':' y despues la 'w' si despues deseas guardar y salir puedes presionar 'wq' esto es para guardar y salir.

Moverse entre Archivos

Si deseas llegar a la definicion de cualquier objeto puedes combinar las teclas 'g' y 'd' Pero si se desea ir directamente al archivo puedes presionar las teclas 'g' y 'f' por ejemplo desde este archivo HTML hasta el archivo style.css Importante: Una vez se guarde este archivo podras navegar hacia otros archivos.

Nota:

Existe una serie de historial cuando se navega entre definiciones o archivos, por lo tanto puedes volver hacia atras con las teclas 'Control'+ 'O'

Eliminar, undo y redo

Para eliminar una palabra dentro del modo de visualizacion se puede hacer presionando la tecla 'd' y la tecla 'w' de 'palabra'

Rehacer

Para rehacer cambios se puede presionar las teclas 'control' y 'r'

Eliminar texto de una Linea completa sin borrar la linea

Se debe presionar las teclas 'd' y '$'

Cortar y pegar

En realidad cuando se elimina una linea con la tecla 'd' dos veces, tecla 'p' esto lo hara despues de la linea donde se encuentra el cursor. Para pegarlo antes se realiza con la 'P' mayuscula.

Remplazar

Este comando la verdad es una porqueria porque solo remplazas una sola letra, pero bueno esto se realiza con la letra 'r'.

Remplazar una palabra

Este comando es mas util porque puedes remplazar una cadena de caracteres. Esto se realiza con la letra 'c' + 'w' esto significa change word aunque remplazara el texto desde la posicion del cursor en adelante.

Remplazar una palabra sin importar la posicion del mouse

Tal como su nombre lo indica puedo remplazar una palabra sin importar la posicion del mouse. Esto se realiza con la combinacion de teclas 'c'+'i'+'w'

Saltos de linea

Para viajar a la primera linea del documento se presiona dos veces la letra 'g' y para ir al final del documento presiona la letra 'G' mayuscula. Tambien para viajar hacia una linea en especifico cuando te encuentres en modo normal, escribe el numero de la linea y presiona la tecla 'G' mayuscula.

Busqueda de palabras

El siguiente comando solo sirve para realizar busquedas despues de la posicion del cursor. Se realiza con las teclas '/'+ La palabra que se desea buscar. Tambien para buscar desde la posicion del cursor hacia atras se realiza desde el modo normal con la tecla '?' + La palabra que estamos buscando. IMPORTANTE: Si tienes muchos resultados puedes navegar entre ellos con la tecla 'n'. Si por alguna razon te pasaste al momento de buscar puedes volver con la tecla 'N' mayuscula.

Viajar entre los parentecis

Se posiciona el mouse en el parentecis o llaves y se presiona el simbolo de porcentaje (%)

Viajar a la primera linea de codigo

Esto se realiza con el numero '0'

Viajar a la ultima linea de codigo

Esto se realiza con el simbolo de '$'

Copiar una linea de codigo

Esto se realiza con la letra 'y' dos veces y si deseas copiar desde la posicion del cursor hasta la ultima linea se realiza con las teclas 'y'+'$'. Para copiar desde la posicion del raton hacia atras se realiza con las teclas 'y'+'0'

Remplazar palabras

Esto se realiza en el modo normal con el comando ':s' seguido de ello escribe la palabra que queremos remplazar con un slash '/the' y despues se escribe la palabra del remplazo '/de'. Al final el resultado es el siguiente: ':s/the/de'. Sin embargo esto solo remplazara la primera ocurrencia, para remplazarlas todas en esa linea donde se encuentra el cursor es necesario al final del comando agregar una '/g'. De esta manera ':s/the/de/g'

Remplazar palabras en todo el Documento

Esto se realiza desde el modo de VISTA con el siguiente comando: '%s/palabra/palabra-remplazo/g' Lo que hace esto es pregutar cada vez que va a remplazar la letra.

Insertar una nueva linea

Esto agregara la nueva linea debajo de donde se encuentra el cursor, se realiza con la tecla 'o' y para agregar texto en la parte superior de donde se encuentra el cursor se realiza con la 'O' mayuscula.

Remplazar con el comando inutil

Si se presiona la letra 'R' mayuscula se remplazara el texto desde donde se encuentra el cursor.

Seleccionar texto en modo visualizacion

Cuando se encuentra en modo normal se presiona la tecla 'v' y se puede seleccionar el texto.

Viajar a la primera linea

Para viajar a la primera linea en el modo de edicion se presionan la tecla 'I' mayuscula.

Volvel al navegador de archivos

Navegar Verticalmente

Se puede realizar de forma relativa con los numeros y la letra 'k' o 'j'
